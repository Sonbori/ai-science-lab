# AI Science Lab

과학·기술 주제를 AI와 구조화된 리서치로 해석해,
사람이 이해하고 활용할 수 있는 결과물로 바꾸는 개인 연구 저장소.

## Goals
- 과학·기술 주제를 짧은 해설문, 원페이지, 미니 리포트로 정리
- AI를 활용해 자료 조사, 구조화, 초안 작성, 비교 분석 수행
- 장기적으로 포트폴리오와 수익화 가능한 작업물 축적

## Repository Structure
- `topics/` : 진행 중이거나 보관된 주제
- `sources/` : 논문, 기사, 데이터셋 관련 자료
- `notes/` : 일간/주간 메모와 아이디어
- `drafts/` : 작성 중인 초안
- `final/` : 최종 결과물
- `data/` : 원본/가공 데이터
- `assets/` : 이미지, 차트, 다이어그램
- `templates/` : 반복 사용 템플릿

## First 30-Day Focus
1. AI는 과학 연구를 어떻게 바꾸는가
2. 즉각적 보상에 약한 사람은 게으른가, 설계 문제인가
3. 인류는 왜 바다를 아직도 잘 모를까
